Projects

  • SquadUp

    Lead a team of 5 developers as the Project Manager. SquadUp is an online community building application that allows users to form and join squads based on the games they love to play. Each squad can stay connected through a dedicated squad chatrooom.

  • BackBurner

    An improved to-do list application that game-ifies task management to keep user motivated. BackBurner grants the user full control over setting up and managing their own to-do list; complete with an Experience Points (XP) System and a Rewards System.

  • Learn Spanish by Spaced-Repetition

    A language learning tool that implements the "spaced repetition learning technique" (Wiki). This implementation of the algorithm allows the user to view their progress as they learn.

  • Trivia Quiz App

    Quiz App is a simple trivia quiz app that obtains its cache of questions from an open source api. Most notably the app tracks a user's high score.
